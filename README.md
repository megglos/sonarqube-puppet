# Sonarqube puppet installation

Install Sonarqube 5.2 using Puppet in a Virtual Machine with Vagrant along with some plugins

## Requirements

* [Download Vagrant](https://www.vagrantup.com/downloads)
* Enable virtualization

## Run

Execute vagrant:

```shell
$ vagrant up
```
